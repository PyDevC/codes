Cpp is not purely object-oriented language byt a hybrid that contains the funcitonality of the C programming langauge.

Features are:
- Universally usable modular programs
- efficient, close to the machine programming
- portable programs for various platforms.
- supports the concept of object-oriented programming

Object oriented programming:
- data abstraction
- data encapsulation for controlled acess to object data
- inheritance by creating derived classes
- polymorphism
